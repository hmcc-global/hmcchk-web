# Button

Usage:

```jsx
import Button from 'ui/src/components/Button';

<Button colorScheme="blue">Learn more</Button>
<Button variant="outline" colorScheme="red">Learn more</Button>
<Button isFullWidth colorScheme="yellow">Learn more</Button>
<Button href="/docs">Go to docs</Button>
```

Props: `variant`, `size`, `colorScheme`, `leftIcon`, `rightIcon`, `isFullWidth`, `isDisabled`, `isLoading`, `as`, `href`, `onClick`.
